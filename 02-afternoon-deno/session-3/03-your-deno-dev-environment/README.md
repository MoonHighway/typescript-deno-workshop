To install Deno, we'll run curl:

```bash
curl -fsSL https://deno.land/install.sh | sh
```

Or if you're on Windows, you can run: 

```
irm https://deno.land/install.ps1 | iex
```

You can also generate a Deno Project from scratch:

```bash
deno init 
```
